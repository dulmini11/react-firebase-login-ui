# React Firebase Login UI

A modern, responsive login application built with React, Material-UI, and Firebase Authentication.

---

## Live Demo

https://login-app-internship-b404b.web.app

---

## Installation

**Clone the repository**
```bash
git clone https://github.com/dulmini11/react-firebase-login-ui.git
cd react-firebase-login-ui
```

**Install dependencies**
```bash
npm install
```

**Run the development server**
```bash
npm run dev
```

---

## Tech Stack

- React 18
- Vite
- Material-UI (MUI)
- Firebase Authentication
- Firebase Hosting

---

## Features

- Email input validation
- Google Authentication via Firebase
- Access token display after login
- Fully responsive design
